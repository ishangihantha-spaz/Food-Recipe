# Gourmet Haven Website

This is a modern, responsive food recipe website built with HTML, CSS (Tailwind via CDN), and JavaScript.

## How to Run

Since you requested **no Node.js**, this website is built to run directly in your browser without any installation or server.

1.  Open the folder `Food Recipe`.
2.  Double-click `index.html` to open it in your web browser (Chrome, Edge, Firefox, etc.).
3.  Navigate through the links to see other pages.

## Features

-   **Home**: Beautiful hero section, featured categories, and trending recipes.
-   **Recipes**: A grid of delicious recipes with filters.
-   **Recipe Detail**: A detailed view for a single recipe (Steak example) with ingredients and instructions.
-   **About**: Our story and team.
-   **Contact**: A working contact form (simulation).
-   **Responsive Design**: Works on mobile and desktop.
-   **Dark/Light Mode**: Optimized for a clean, light gourmet aesthetic.

## Technologies Used

-   HTML5
-   CSS3 (Tailwind CSS CDN + Custom Styles)
-   JavaScript (Vanilla)
-   Font Awesome (Icons)
-   Google Fonts (Inter, Playfair Display)
